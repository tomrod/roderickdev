"AI discovered wholly new proteins before it count count the Rs in 'strawberry,' which makes it neither vaporware nor a demigod but a secret third thing." - Jasmine Sun

Neat quote, and a bit apprehensive. AI is a tool -- a mundane word to most ears, but look at how tool use set humans up for some pretty neat lives. 

Now we can do tasks faster. Not necessarily better (overall quality still improving) -- but the speed improvement gives more time for focusing on quality.

AI isn't a crystal ball, it is a metal detector.
